# IBM Cloud Catalog - Power Virtual Server for ORACLE Single Instance: 'Oracle ready PowerVS - Private Cloud'

# Summary
This deployable architecture is designed to assist you in deploying Oracle database software landscapes into IBM Power Virtual Server - Private Cloud infrastructure.
## Summary Outcome: 
   Creates and prepares Power Virtual Server instances for Oracle databse workloads. This provisions Red Hat Enterprise Linux as the Ansible controller and AIX virtual machines as Oracle instance, retrieves Oracle software from IBM Cloud Object Storage, and performs a guided GRID and RDBMS installation and creates Oracle multitenant database based on user-defined parameters

## Summary Tasks

- Creates and configures one PowerVS instance for Oracle Database based on best practices.
